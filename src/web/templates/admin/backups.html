How to restore from a backup
Each backup file is a complete standalone SQLite database. To restore:
  1. Stop the app (Ctrl+C in the make web terminal).
  2. Copy the live DB out of the way (just in case):
    cp data/copilot.db data/copilot.db.before-restore
  3. Copy the backup into place:
    cp data/backups/copilot-YYYYMMDD-HHMM.db data/copilot.db
  4. Remove any stale WAL/SHM files so SQLite reads the fresh main file:
    rm -f data/copilot.db-wal data/copilot.db-shm
  5. Start the app again: make web
