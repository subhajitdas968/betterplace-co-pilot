▸ Advanced · Named tunnel with custom domain (e.g. copilot.betterplace.co.in)
For a permanent URL under your own domain (requires Cloudflare account):
  1. cloudflared tunnel login — opens browser, pick the betterplace.co.in zone.
  2. cloudflared tunnel create copilot — creates the tunnel. Credentials saved under ~/.cloudflared/<UUID>.json.
  3. cloudflared tunnel route dns copilot copilot.betterplace.co.in — adds the DNS record.
  4. Create ~/.cloudflared/config.yml with: 
    tunnel: copilot
credentials-file: ~/.cloudflared/<UUID>.json
ingress:
  - hostname: copilot.betterplace.co.in
    service: http://127.0.0.1:8000
  - service: http_status:404
  5. Run: cloudflared tunnel run copilot
  6. Wire Cloudflare Access on the route to require @betterplace.co.in Google login — defense in depth on top of our app-level auth.
