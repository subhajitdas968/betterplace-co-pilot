One-time setup outside this UI
Cloudflare and Google can't be configured purely from this app. Two things to do once after you start the tunnel:
a. Add Google OAuth redirect URI
Open Google Cloud Console → APIs & Services → Credentials, find your OAuth 2.0 Client, and add this Authorized redirect URI:
https://{{ named.hostname or suggested_host }}/auth/callback
This is permanent — you'll never have to change it again, unlike the rotating Quick Tunnel URL.
b. (Recommended) Add Cloudflare Access — defense in depth
Open Cloudflare One dashboard → Access → Applications → Add an application → Self-hosted, then:
  1. Application name: BetterPlace Co-Pilot
  2. Subdomain: copilot · Domain: betterplace.co.in
  3. Add a policy: Action Allow, Selector Emails ending in = @betterplace.co.in
  4. (Optional) Require Google as identity provider
Cloudflare will then challenge every request with its own login before it even reaches our app — useful if our app-level auth ever has a bug. You can layer this in/out without restarting the tunnel.
